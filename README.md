# ExZstd

Elixir binding of the Zstandard library

## Installation

The package can be installed by adding `zstd` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:ex_zstd, "~> 0.1.0"}
  ]
end
```

The docs can be found at [https://hexdocs.pm/ex_zstd](https://hexdocs.pm/ex_zstd).
